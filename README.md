# Vue Money Manager

## Project setup
```
npm install
```
```
cd client
npm install
```

### MongoDB Setup
```
replace "your_mongodb_connection_url" with your mongodb url connection 
in server/routes/api/expenses.js and server/routes/api/income.js 
```

### Run
```
npm run dev
```
```
cd client
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
